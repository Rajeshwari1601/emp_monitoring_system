# Frontend Structure

This frontend is designed as a simple Admin Dashboard.

## Directories

- `css/`: Stylesheets.
- `js/`: Application logic.
- `assets/`: Images and icons.
- `components/`: Reusable UI components (HTML snippets or JS classes).
- `services/`: API interaction services.
- `views/`: Specific page logic if needed.

## Setup

Open `index.html` in a modern web browser.
